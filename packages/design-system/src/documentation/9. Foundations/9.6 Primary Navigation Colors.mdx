## Why These Colors Fit Together

1. **Deep anchor background (`#052F35`)** creates a stable left-rail shell and visually separates navigation from work content.
2. **White text variants** provide clear hierarchy without introducing extra hues.
3. **Teal focus/accent (`#27C7AC`)** gives strong state visibility against dark background for hover, active icons, and focus rings.
4. **Bridge to token system**:
   - `#052F35` aligns with `bcolor.primary.1000`.
   - `#15808C` and `#449CA7` align with product semantic theme colors used elsewhere.
5. **Neutral workspace handoff** (`#F1F2F4`) keeps content surfaces readable while nav remains distinct.

## Recommendation

Promote nav-specific semantic aliases (for example `ucolor.nav.background`, `ucolor.nav.focus`, `ucolor.nav.text.secondary`) so application shells avoid hardcoded values and stay themeable.
