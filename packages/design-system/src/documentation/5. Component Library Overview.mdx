#### UniCard Variants

- `default`: Standard Ant-style card for generic content.
- `project-overview`: GuidedFlow project status card style.
- `tool-registry`: Tools catalog card with version, functions, and usage metadata.
- `platform-component`: Resource/component card with metric highlight and CTA.
- `spotlight`: Featured journey card with premium gradient treatment.
- Color configuration: `cardColorConfig` supports color-pair lists (`300`/`700`), manual or random assignment, and apply target (`header`, `card`, `none`).
- Extension model: add future variants by extending `UNI_CARD_VARIANTS` and adding a renderer in `packages/core/src/components/card/UniCard.tsx`.
