# Tools & Process
---
<br/>

### Tools:
#### Design:

1. Figma - Design and Token Management
2. Ant Design - Base Design System

#### Development:

1. React - Base JavaScript Framework
2. Ant Design React - Base Component Library
3. AG Grid - Base Library for Building Tables
4. Tailwind CSS - Utility-First Styling
5. ECharts - Base Library to Construct Various Charts
6. Storybook - Component Library
7. Jira / Issues on Github - Bug Report and Feature Request

#### Documentation:

1. Figma - Component Documentation
2. Notion OR Atlassian - Roadmapping and Project Planning
3. Jira OR Spreadsheets OR TypeForm - Priorities and Request Form

### Process:

1. With the Design System Team, we map out the roadmap for the Design System as we receive requests.
2. Design the comp component and publish it to the designers to use in Figma.
3. Document the user stories, states, token usage, dimension, and library reference for the component.
4. Once the documentation is reviewed and approved, an engineer will be assigned to develop the component.
5. Run the developed component through an automated workflow to unit test and smoke test.
6. Visual and functional QA the component with the Design System Team.
7. Publish the updated framework and write up a version update for documentation.
