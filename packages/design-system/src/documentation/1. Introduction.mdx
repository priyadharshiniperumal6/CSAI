# Introduction
---
The Universal Theme Framework is a powerful UI component library built for Uniphore products.
It combines the best of [Ant Design](https://ant.design), [Tailwind CSS](https://tailwindcss.com), [AG Grid](https://www.ag-grid.com), and [Apache ECharts](https://echarts.apache.org/en/index.html) to provide a comprehensive set of tools for building modern, responsive web applications.
Version: <br />
Latest {version} - {lastUpdate}

---
### Key Features:
1. Pre-built UI "atom-sized" components based on the [Ant Design System for Figma](https://www.antforfigma.com)
2. Utility-first CSS with Tailwind CSS:<br/>
  a. Provides the flexibility to scale<br/>
  b. Easier to manage and organize for a enterprise-level software development
3. Advanced data grid functionality, powered by AG Grid
4. Utilizes React as the base framework
5. Apache Echarts for Visualizations
---
### Prerequisites:
Before your begin, please make sure you have the following items installed:
1. Node.js (Version 20 or later)
2. Pnpm (Version 9 or later)
3. Git
